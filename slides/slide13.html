Attribute directives

Built in Angular2 directives that modify elements in a template

  • Angular2 comes with some built in directives for modifying the styles of elements in component templates
  • These are generally similar to AngularJS built in directives but with different syntax

Attribute directives

        
            <div [ngClass]="setClasses()">This div is saveable and special</div>
        
    
        
            setClasses() {
              let classes =  {
                saveable: this.canSave,
                special: this.isSpecial,
              }
              return classes;
            }
  • ngClass allows a component to conditionally apply different classes to an element in its template
  • the setClasses function is defined in the component and returns an object with class names and true/false

Attribute directives

        
            <div [ngStyle]="setStyles()">This div is italic, normal weight</div>
        
    
        
            setStyles() {
              let styles = {
                'font-style':  this.canSave      ? 'italic' : 'normal',
                'font-weight': !this.isUnchanged ? 'bold'   : 'normal'
              }
              return styles;
            }
  • Very similar to the ngClass directive, ngStyle allows a component to conditionally apply different CSS styles to an element in its template
  • the setStyles function is defined in the component and returns an object with CSS attribute names and their corresponding values
