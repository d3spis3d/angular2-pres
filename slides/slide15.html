Components

      
          @Component({
            selector:    'hero-list',
            templateUrl: 'app/hero-list.component.html',
            directives:  [HeroDetailComponent],
          })
          export class HeroesComponent {
            static get parameters() {
                return [[HeroService]]
            }

            constructor(_service){
              this.heroes = this._service.getHeroes();
            }

            selectHero(hero) {
              this.selectedHero = hero;
            }
          }
  • So here is what a Component looks like in Angular2
  • We have a Component decorator that declares the DOM selector, templateUrl, and child Component directives of this Component
  • The Component itself is an ES6 class
