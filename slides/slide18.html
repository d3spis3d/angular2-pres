Services

  • Handle most of the business logic of Angular2 apps
  • Communicate with the backend to load/store data
  • Much the same of AngularJS services

Services

        
            @Injectable
            export class HeroService {
                static get parameters() {
                    return [[BackendService]];
                }

                constructor(backendService) {
                    this._backend = backendService;
                }

                getHeroes() {
                    return this._backend.get('/api/v1/heroes');
                }
            }
